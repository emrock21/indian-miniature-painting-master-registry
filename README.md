# Indian Miniature Painting Master Registry

This registry documents India’s miniature painting traditions, including Mughal, Rajput, Pahari, Deccan, Tanjore, and Mysore schools.

## Purpose

To preserve India’s miniature painting heritage on-chain in a permanent, verifiable, and censorship-resistant format.

Each entry records:

- region  
- school or lineage  
- style name  
- pigments  
- techniques  
- iconography  
- cultural uniqueness  

## Why this registry matters

India’s miniature painting schools include:

- Mughal Miniature (Delhi–Agra)  
- Mewar & Kishangarh Rajput Schools (Rajasthan)  
- Kangra & Basohli Pahari Schools (Himachal Pradesh & Jammu)  
- Deccan Miniature (Bijapur, Golconda)  
- Tanjore Painting (Tamil Nadu)  
- Mysore Painting (Karnataka)  
- Pattachitra (Odisha)  

Each tradition uses distinct pigments, brushwork, iconography, and aesthetic systems.
